# ID Card Upload & Advanced Admin Powers Requirement

## ID Card System

Student apna ID card upload nahi karega.

Sirf:

* Admin
* Teacher/Staff

ID card upload/manage kar sakte hain.

### ID Card Features

* Admin/Teacher uploaded ID card hi student dashboard mein visible hona chahiye
* Student sirf:

  * View
  * Download
  * Print

kar sake.

## ID Card Design

* Same-to-same screenshot design
* Sirf barcode use hoga
* QR code nahi hoga
* Provided logo use hoga
* Background image admin change kar sakega

---

# Advanced Admin Powers

Admin ko full super-admin powers deni hain taki pura portal control kar sake.

## Admin Powers List

### Student Control

* Add/Edit/Delete Students
* Student login block/unblock
* Student profile edit
* Student documents manage
* Student ID card upload/change
* Student signature manage

---

### Teacher Control

* Add/Edit/Delete Teachers
* Teacher permissions manage
* Teacher subjects assign
* Teacher login manage

---

### Subject Management

* Add/Edit/Delete Subjects
* Semester-wise subjects assign
* Subject teachers assign

---

### Fees Management
