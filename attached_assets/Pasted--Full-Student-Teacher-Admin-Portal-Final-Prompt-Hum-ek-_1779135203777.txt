# Full Student, Teacher & Admin Portal Final Prompt

Hum ek complete production-level Student, Teacher/Staff aur Admin Portal bana rahe hain jo MongoDB database ke saath fully dynamic aur fully working hona chahiye. Pura portal screenshots, photos aur assets ke according same-to-same design mein banana hai jo main provide karunga.

---

# Login System

## Separate Login URLs

* Student Login → `/login`
* Teacher/Staff Login → `/staff`
* Admin Login → `/admin`

Normal login sirf students ke liye open hoga. Staff aur Admin login sirf direct URL se open honge.

## Login Features

* ID/Email & Password Login
* Forgot Password
* Secure Authentication
* MongoDB Authentication
* Role-Based Redirects
* Working Demo Credentials
* Mobile Responsive Login UI

Main jo login screenshots aur credentials dunga wahi same design aur same credentials use karna hai.
